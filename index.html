Подбери профессию за 30 секунд!

Перед вами представлен продукт с помощью которого можно подобрать профессию на основе ваших интересов.

Инструкция по применению:

Выберите в каждой группе по одному фильтру.

При этом в группе "Soft skills" есть возможность выбора двух фильтров, сюда не входит фильтр "Не учитывать". Выбирая фильтр "Не учитывать", пропадает возможность выбора оставшихся двух.

Можете фильтровать профессии по одной интересующей группе, для этого выберете в остальных "Не учитывать".
